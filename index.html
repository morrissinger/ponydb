PonyDB

PonyDB is an in-memory graph database for node.js, based on HeliosJS.

In order to traverse the graph, PonyDB uses a Gremlin inspired graph traversal language to query, analyze and manipulate the graph. Gremlin is an open source project maintained by TinkerPop. For more information on Gremlin, see the Gremlin wiki.
